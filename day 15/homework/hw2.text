2. ახსენით რისთვის გამოიყენება value ატრიბუტი option თეგთან, ასევე ახსენით რა დანიშნულება აქვს <select> და <option> თეგებს


value ატრიბუტი გამოიყენება <option> თეგთან, რათა განსაზღვროს ამ არჩევანის მნიშვნელობა, რომელიც გადაიგზავნება ფორმის გაგზავნისას.
<select> თეგი HTML-ში გამოიყენება ჩამოსაშლელი (dropdown) მენიუს შესაქმნელად, რომელიც მომხმარებელს აძლევს საშუალებას აირჩიოს ერთი ან რამდენიმე ვარიანტი.
<option> თეგი გამოიყენება select-ის შიგნით, რათა ჩამოთვალოს შესაძლო არჩევანი. თითოეული <option> წარმოადგენს ერთ ელემენტს ჩამოსაშლელ მენიუში.
